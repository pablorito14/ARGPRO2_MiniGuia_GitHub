Mini-Guia GitHub

1) Entrar en GitHub y crear un repositorio nuevo. Poner un nombre (y descripción si se quiere), dejar en publico (para poder desplegar en Pages) y crear

2) Vamos a ver los comandos que vamos a tener que ejecutar en la consola. Se ejecuta cada linea por separado. La primera linea es simplemente un mensaje y no es necesario ejecutar. Prestar atencion a que la 3ra linea, en lugar de usar git add README.md tenemos que poner git add . para que tome todos los archivos de nuestro proyecto

3) En vs code dar click derecho sobre la carpeta de nuestro proyecto y elegir Abrir en terminal integrado y esperar a que en la consola salga la carpeta que queremos subir. Prestar atención que este dentro de esa carpeta, sino va a haber problemas.
